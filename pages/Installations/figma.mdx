## **Récupération  des variables figma sur vs code**
1. Extension pour récuperer le json des variables du figma ⇒ [Variables to Json](https://www.figma.com/community/plugin/1301567053264748331/variables-to-json?searchSessionId=lz70mrpn-hwxsirfkwlu)
2. Transformer le json en variable css ⇒  https://moon-moon.fr/pages/json-to-css
3. Copier dans le fichier mm-variables.css 
4. Vérifier et ajuster les class des fichiers css (supprimer ou ajouter)
