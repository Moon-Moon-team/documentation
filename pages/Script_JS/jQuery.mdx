# Intégrer jQuery à votre thème

***

## Version 3.7.0

### Étape 1 : Ajouter le code dans le head du fichier 'theme.liquid'

```jsx
<!-- jQuery -->
<script src="{{ 'jquery-3.7.0.min.js' | asset_url }}" defer></script>
```

### Étape 2 : Créer un asset 'jquery-3.7.0.min.js'

Créez un nouveau fichier dans le dossier `assets` de votre thème avec le nom `jquery-3.7.0.min.js` et copiez-y le code jQuery 3.7.0 minifié.

Vous pouvez télécharger la version minifiée depuis le site officiel de jQuery : https://jquery.com/download/
