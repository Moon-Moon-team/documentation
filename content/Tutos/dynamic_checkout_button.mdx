# Dynamic Checkout Button

## Dans le product form : 

``` Liquid
{{ form | payment_button }}
```

> Attention : il faut un (et un seul) input avec name=”id” ayant la variant id en value.

> Attention : Si on utilise de l’abonnement, il faut un (et un seul) input avec name=”selling_plan” ayant le selling plan id en value (par defaut la value doit être vide si on sélectionne un plan one shot par defaut).

## Utilisation dans le panier : 

``` Liquid
{{ content_for_additional_checkout_buttons }}
```
